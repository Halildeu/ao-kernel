## Final Plan — 6 Alt İş

### B3a — Lossless ExtensionManifest

`ExtensionManifest` dataclass'ı schema required alanların tümünü + discovery-önemli opsiyonel alanları taşısın:

- **Required (schema):** `version`, `extension_id`, `semver`, `origin`, `owner`, `layer_contract`, `entrypoints`, `policies`, `ui_surfaces`, `compat`
- **Optional (discovery/provenance):** `enabled`, `gates`, `docs_ref`, `ai_context_refs`, `tests_entrypoints`, `model_kind`, `outputs`
- **Computed (yeni):** `manifest_path` (source location), `content_hash` (sha256 of raw JSON)

### B3b — Schema validation

- `jsonschema.validate(data, extension_manifest_schema)` her manifest için
- JSON parse error → warn + skip (mevcut davranış korunuyor)
- Schema invalid (required eksik, enum mismatch) → warn + skip (**YENİ** — sessizce default'lamayı bırak)
- `load_from_*` dönüş değeri `(loaded_count, skipped: list[dict])` — skip nedeni kayıt altında

### B3c — Duplicate conflict resolution

- Aynı entrypoint iki manifest'te → **first-wins** (sıralı load)
- Workspace override bundled'dan sonra yüklenir → workspace kazanır (istenen davranış)
- Bundled içinde duplicate → **warning log** + ilk gelen kazanır (non-deterministic order'ı engellemek için `sorted(path)` garanti)
- Registry'e yeni query: `find_conflicts() -> list[dict[entrypoint, declarers]]`

### B3d — Compat filter

- Manifest `compat.core_min` / `compat.core_max` → `ao_kernel.__version__` ile karşılaştır
- `packaging.version` kullan (stdlib değil ama jsonschema gibi core)
- Mismatch → manifest `_activation_blockers: ["compat:core_min=X.Y > current=A.B"]`
- Blocked manifest `list_enabled()` çıkışında **görünmez**, `list_all()` çıkışında görünür + blocker bilgisi

### B3e — Workspace root contract + stale refs

- **Kesinlik:** Loader **project_root** kabul eder (`.ao` dizinini **içeren** dizin)
- `client._workspace_root` zaten project_root → direkt geçer
- Auto-discover path (`config.workspace_root()` `.ao` döner) → `.parent` normalize
- B4'teki `_find_workspace_root()` pattern'i `ao_kernel/workspace.py`'a taşınır (shared helper)
- **Stale refs flag:** Manifest `ai_context_refs` + `docs_ref` + `tests_entrypoints` path'leri kontrol et; var olmayanlar `_stale_refs: list[str]` alanına yazılır (warning, not blocker)

### B3f — Explicit activation dispatch

Codex'in önerisi (b): code-owned dispatch, D7 korunur.

- **Yüzey seçimi:** `kernel_api_actions` (policy allowlist + request schema enum'da zaten canlı)
- **Yeni modül:** `ao_kernel/extensions/dispatch.py`
  - `ActionRegistry` — explicit `register(action_name, handler, extension_id)`
  - `resolve(action_name) -> (handler, extension_id) | None`
  - `list_actions() -> list[(action, extension_id)]`
- **Hello extension:** `ao_kernel/defaults/extensions/PRJ-HELLO/extension.manifest.v1.json` (yeni) + `ao_kernel/extensions/handlers/prj_hello.py` (yeni, `hello_world` action handler)
- **Client integration:** `AoKernelClient.extensions: ExtensionRegistry` property; `AoKernelClient.call_action(action, params)` public API
- **MCP integration:** Yeni tool **veya** mevcut `ao_llm_route` üzerinden — B3f scope'u sadece client-side; MCP wiring follow-up olabilir (scope fence burada)
