## Final Plan — 6 Alt İş (Tek PR)

### B5a — `record_decision` semantik fix

`ao_kernel/context/agent_coordination.py:record_decision`:
- `auto_promote=True` + `confidence >= promote_threshold` → `promote_decision` (canonical, full TTL)
- **Aksi halde → session context'e `upsert_decision`** (ephemeral, canonical'a YAZMA)
- `fresh_days=7` silent canonical path kaldırılır

### B5b — `end_session` + `finalize_session_sdk` birleştir

`session_lifecycle.end_session`:
- Yeni parametreler: `auto_promote: bool = True`, `promote_threshold: float = 0.7`
- Tek yerde promote, backward compat (default eski davranış)

`agent_coordination.finalize_session_sdk`:
- Sadece `end_session(context, workspace_root, auto_promote=auto_promote, promote_threshold=promote_threshold)` çağırır
- Kendi `promote_from_ephemeral` kaldırılır → double-promotion düzelir

### B5c — Revision hash

- `get_revision`: `sha256(...).hexdigest()` — tam 64 char
- Test'lerde `assert len(rev) == 16` yerine `assert len(rev) > 0` (opaque)

### B5d — `check_stale` advisory

- Docstring: "ADVISORY ONLY. Filesystem lock / CAS planned in separate CNS. Concurrent writes between check_stale() and write() can produce lost updates."
- Follow-up CNS referansı

### B5e — Facade expose

**`ao_kernel/context/__init__.py`:**
```python
from ao_kernel.context.agent_coordination import (
    record_decision, query_memory, get_revision,
    read_with_revision, check_stale,
    compile_context_sdk, finalize_session_sdk,
)
__all__ = [..., "record_decision", "query_memory", ...]
```

**`AoKernelClient`** wrapper metotları:
- `client.record_decision(key, value, ...)` — `self._session_id` otomatik
- `client.query_memory(pattern, category)` 
- `client.get_revision()`
- `client.read_with_revision(pattern, category)`
- `client.has_changed(last_revision)` — yeni isim (check_stale alias korunur)
- `client.compile_context_sdk(...)` — LLM çağrısı yapmadan preamble
- `client.finalize_session(auto_promote, promote_threshold)` — birleşik

### B5f — Testler
