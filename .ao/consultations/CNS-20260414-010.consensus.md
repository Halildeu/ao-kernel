## Final Plan v4 (4 düzeltme ile v3)

### Düzeltme 1: ToolSpec.allowed=True + handler-level deny envelope

**Yanlış (iter-3'te Claude önerdi):**
```python
ToolSpec(name="ao_memory_read", allowed=False, handler=...)
# handler içinde explicit gate → BYPASS imkansız
```

**Doğru (iter-3 blocking-1):**
```python
ToolSpec(name="ao_memory_read", allowed=True, handler=handle_memory_read)
# handler içinde policy_mcp_memory load + enabled check + deny envelope
```

Gateway `authorize()` `TOOL_NOT_REGISTERED/NOT_ALLOWED` early return yapar; handler hiç çalışmaz. Dolayısıyla gate handler içinde olmalı.

### Düzeltme 2: Param-aware workspace resolver

**Yanlış (iter-3'te Claude önerdi):**
```python
# _with_evidence() + implicit promote:
ws = _find_workspace_root()  # global CWD'den çözer
```

**Doğru (iter-3 blocking-2):**
```python
def _resolve_workspace_for_call(params: dict, *, fallback=None) -> Path | None:
    \"\"\"Param-aware workspace: tool çağrısı workspace_root geçebilir.\"\"\"
    if isinstance(params.get("workspace_root"), str):
        ws = Path(params["workspace_root"])
        if ws.is_dir():
            return ws
    return fallback() if fallback else _find_workspace_root()
```

Memory tool'u workspace_root parametresi geçince evidence/promotion o workspace'e yazar. Global CWD fallback sadece param yoksa.

### Düzeltme 3: Implicit promote generic MCP/tool policy (NOT memory-scoped)

**Yanlış (iter-3'te Claude önerdi):**
```json
// policy_mcp_memory.v1.json
{"implicit_canonical_promote": {...}}
```

**Doğru (iter-3 warning):**
```json
// policy_mcp_tool_calling.v1.json (veya yeni policy_mcp_implicit_promote.v1.json)
{
  "implicit_canonical_promote": {
    "enabled": true,
    "threshold": 0.8,
    "source_prefix": "mcp:tool_result"
  }
}
```

Çünkü `call_tool()` ALL MCP tool sonuçları için çalışıyor, sadece memory değil.

### Düzeltme 4: CAS API yüzeyi minimal

**Yanlış (iter-3'te Claude önerdi):**
```python
# Ayrı public fonksiyonlar
save_store_cas(...)
promote_decision_cas(...)
forget_cas(...)
```

**Doğru (iter-3 Codex):**
```python
# Tek yeni public low-level + mevcut fonksiyonlarda opsiyonel keyword
save_store_cas(ws, store, *, expected_revision, allow_overwrite=False)  # YENİ public

# Mevcut public fonksiyonlar opsiyonel keyword kazanır (backward compat):
promote_decision(..., expected_revision=None, allow_overwrite=True)
self_edit_memory.forget(..., expected_revision=None, allow_overwrite=True)

# İç helper:
_mutate_with_cas(ws, mutator_fn, *, expected_revision, allow_overwrite)  # private
```

`remember`, `update`, `recall` zaten `promote_decision` üzerinden kazanır. `*_cas` duplicate'ı yok.
